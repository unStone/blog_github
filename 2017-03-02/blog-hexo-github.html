开始利用hexo搭建blog

安装

  • 打开node，输入以下命令（随便在哪个目录下）
    npm install hexo -g
    淘宝镜像（可选）
    cnpm install hexo -g
    yarn（可选）
    yarn global add hexo

上面三种命令得到的结果是一样的，不同的只是你安装包的习惯

构建hexo项目

  • 等hexo全局安装完成后，我们就可以开始我们的博客搭建之旅了
  • 首先打开node，cd到你想要博客所在的目录位置，然后输入以下命令
    hexo init 目录名
  • 例如

  • 等待运行结束后，hexo就已经帮你把基础的blog搭建好了，是不是很快呀，别急我们还有一点东西没弄好

安装依赖

  • 构建好项目后就是安装程序的依赖了

  • 在刚刚构建的目录下，输入以下命令（随便在哪个目录下）
    npm install
    淘宝镜像（可选）
    cnpm install
    yarn（可选）
    yarn

  • hexo3.0开始与以前的版本有区别，想要本地运行得先安装另外一个模块 hexo-server才行，输入以下命令安装
    npm install hexo-server --save
    淘宝镜像（可选）
    cnpm install hexo-server --save
    yarn（可选）
    yarn add hexo-server

  • 现在开始我们的博客就正式生成啦

本地试运行

  • 上述步骤做完后，你一定迫不及待的想看看博客是什么样的了吧！来来来跟我一起在node中输入以下命令
    hexo server
  • 然后在浏览器地址栏中输入 localhost:4000 你闪亮的博客就登场啦~~

上线才是王道

  • 那人类的欲望是无止境的，既然本地可以运行了，那就一定想把它部署到线上喽！跟着我做把代码发布到线上！
  • 首先打开代码编辑器，在这里强烈推荐vscode 室友推荐的，我很中意
  • 打开刚刚搭建好的博客的文件夹，找到_config.yml并打开该文件修改如下设置
    deploy: type: git repo: 你的项目ssh地址 branch: master
  • 修改配置保存后，依次输入以下命令
    hexo clean
    hexo generate
    hexo deploy
  • 到这里线上的博客也搭建好了，在浏览器中输入你github上的地址看看吧
